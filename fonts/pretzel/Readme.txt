Pretzel Font

The Pretzel font is a rough-edged font based on a modified version of Linux Libertine Bold. Like all other JLH Fonts, this font is in the public domain, and it comes in TrueType and OpenType formats. It contains upper-case and lower-case letters, old-style figures, numbers, some punctuation characters and some mathematical symbols.
